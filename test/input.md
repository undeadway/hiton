# HITON 外加功能

## 颜色、字号、注音

### 颜色

#[FF0000]{==血字的研究==}

### 字号

?[48px]{_48px_ 大小的文字}  
?[20ps]{ps不是合法的单位，所以用**px**的替代}

### 注音

::[一]{yi}::[个]{ge}::[一]{yi}::[个]{ge}::[字]{zi}::[地]{de}::[注]{zhu}::[音]{yin}

### 上下标

!!CO~+{2}!!  
C~+{6}H~+{12}O~+{6}  
y = x~^{2}

### 文字对齐

>>居中对齐<<

|:左对齐<<

>>右对齐:|

### 预定义段落

  这里可以写	预定义结构
  不用考虑 !!md!! 的格式
]]
